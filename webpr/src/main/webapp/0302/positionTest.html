position : static 기본값, 코딩순서대로 배치

position : relative 상대위치, static(기본 값: 현재위치)을 기준으로 재배치

position: relative;
top: 100px;
left: 100px;

position : absolute 절대위치, 컨테이너(기본 값: 홈페이지 왼쪽 맨위)를 기준으로 재배치

position: absolute;
top: 200px;
left: 150px;

position : fixed 고정위치, 컨테이너를 기준으로 재배치

position: fixed;
top: 300px;
right: 100px;
